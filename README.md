# 23-1_IAIP_Final

Upbit API를 사용하여 암호화폐 가격을 예측하여 자동으로 실시간 매매를 진행하는 코드입니다.

사용한 딥러닝 모델은 시계열 데이터의 특성과 과거의 데이터를 이용하여 예측하는 LSTM모델을 사용하였습니다.
